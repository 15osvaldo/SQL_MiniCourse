Relacionales y no relacionales (NoSQL)

Relacionales:
Conjunto de datos que estan organizados y almacenados de manera sistematica en un conjunto de tablas descritas y bien organizadas para un uso posterior de los datos.

NoSQL:
Conjunto de datos que pueden ser consultados y analizados desde diferentes fuentes de datos que no necesariamente utilizan el lenguaje SQL para realizar una consulta sobre los datos.

Definición de SQL server

Gestor de base de datos creado por Microsoft corporation para el almacenamiento modificacion y extraccion de la informacion de la base de datos.

El lenguaje utilizado para el manejo de datos es Transact-SQL o T-SQL en su abreviada para la escritura de sentencias que utiliza el gestor.

Que son los datos:
Es la unidad minima de información la cual es representada por medio de letras, numeros simbolos, imagenes y otras formas de datos.

Analisis de datos:
Proceso de estudio que consiste en limpiar, inspeccionar y transformar los datos de manera que puedan convertirse en informacion util.

Manejo de datos:

Sirve para analizar la información que tenemos a mano, la cual nos guiara a tomar mejores decisiones, sacar mejores concluciones acerca de diversos tópicos que son cubiertos con los datos que tenemos almacenados en nuestra base de datos.

1. Sacar mejores conclusiones
2. Realizar mejores tomas de decisiones
3. Generar mejores reportes
4. Generar mejores consultas
5. Tener más opciones a la mano

Definición de T-SQL

Es un lenguaje de programación de base de datos relacionales que sirve para crear objetos en base de datos y manejar datos, T-SQL es una extensión del estándar SQL para trabajar con base de datos. Es un lenguaje procedural utilizado para programar stored procedures y tiggers en base de datos.

1. Consultas
2. Edición
3. Inserción
4. Eliminación


DDL (lenguaje de definicion de datos):
Ayuda a cambiar la estructura de una base de datos.

Sentencia CREATE

+


                                    Sentencia utilizada para crear una base de datos.
                                    
                                        CREATE DATABASE Nombre_DB;

Creación de una tabla.

+


                                    Ejemplo de creación de una tabla.
                                    
                                        CREATE TABLE Nombre_Tabla (
                                            codigo INTERGER,
                                            Nombre VARCHAR(100),
                                            Apellido VARCHAR(100)
                                            );

Sentencia ALTER

+


                                    Sentencia utilizada para modificar una base de datos.
                                    
                                    -> En este ejemplo se agrega una columna.
                                    
                                        ALTER TABLE Nombre_Tabla ADD codigo INT;

Sentencia DROP

+


                                    Sentencia utilizada para eliminar una columna de
                                    una base de datos.
                                    
                                       DROP TABLE Nombre_Tabla;

Sentencia TRUNCATE

+


                                    Se utiliza para eliminar todos los registros 
                                    de una tabla sin afectar su esquema y estructura
                                    
                                        TRUNCATE TABLE table_name;
DML (lenguaje de manipulacion de datos):
Son un subconjunto de SQL que se utiliza para insetar, actualizar y eliminar datos de una base de datos.

Sentencia SELECT

+


                                Se utiliza para recuperar datos de una o
                                varias tablas de la base de datos. 
                                Seleccionar todos los datos de la tabla customers
                                
                                    SELECT * FROM customers;

Sentencia INSERT

+


                                Se utiliza para insertar nuevos registros en 
                                una tabla de la base de datos. 
                                Insertar un nuevo registro en la tabla customers
                                
                                   INSERT INTO customers
                                    (id, name, email) 
                                    VALUES (1, 'John Smith', 'john@example.com');

Sentencia UPDATE

+


                                Se utiliza para modificar los datos ya 
                                existentes en una base de datos.

                                Actualizar el correo electronico
                                del cliente con ID 1.
                                
                                    UPDATE customers 
                                    SET email = 'john.smith@example.com' 
                                    WHERE id = 1;

Sentencia DELETE

+


                                Se utiliza para eliminar registros de una
                                tabla de la base de datos.
                                Eliminar el registro del cliente con el ID 1.
                                
                                   DELETE FROM customers WHERE id=1;
